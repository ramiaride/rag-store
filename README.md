# MainPage Component

This folder provides a simple React component and CSS for a modern main page with a top bar and a large title "RAG Store".

## Files

- `src/components/MainPage.jsx` - React component (JSX)
- `src/components/MainPage.css` - Component styles

## Usage

1. Ensure you have a React project (Create React App, Vite, etc.).
2. Copy the files into your project at `src/components/`.
3. Import and use in your `App.jsx`:

```jsx
import React from "react";
import MainPage from "./components/MainPage";

export default function App() {
  return <MainPage />;
}
```

4. Run your dev server. CSS is imported inside the component; your bundler must support CSS imports.
